# Roulette

![demo](https://raw.github.com/wiki/takumac/Roulette/images/add_item.gif)

## Overview
This application is a simple roulette application.

## Requirement
- Xcode 12.4
- Run iOS Simulator or iOS Devices
- iOS 13.0 and Later
- Charts 3.6.0
- Eureka 5.3.3
- ColorPickerRow
- MaterialComponents
- RealmSwift

## Usage
```
$ git clone https://github.com/takumac/Roulette<br>
$ pod install
```
